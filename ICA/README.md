# Independent Component Analysis (ICA)

This folder contains Jupyter notebooks implementing Independent Component Analysis (ICA) for feature extraction on the UNSW-NB15 dataset.

## Files

- UNSW-ICA-10.ipynb – ICA with 10 independent components
- UNSW-ICA-20.ipynb – ICA with 20 independent components
- UNSW-ICA-30.ipynb – ICA with 30 independent components

## Objective

Evaluate the impact of different numbers of independent components on Network Intrusion Detection System (NIDS) performance.
