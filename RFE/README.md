# Recursive Feature Elimination (RFE)

This folder contains notebooks implementing Recursive Feature Elimination (RFE) for feature selection using the UNSW-NB15 dataset.

## Files

- IDS_UNSW_RFE_10_20.ipynb
- IDS_UNSW_RFE_30.ipynb

## Objective

Select the most relevant features by recursively eliminating less important features to improve machine learning performance.
