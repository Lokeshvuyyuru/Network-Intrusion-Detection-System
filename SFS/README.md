# Sequential Feature Selection (SFS)

This folder contains notebooks implementing Sequential Feature Selection (SFS) on the UNSW-NB15 dataset.

## Files

- IDS_UNSW_SFS_10.ipynb
- IDS_UNSW_SFS_20.ipynb
- IDS_UNSW_SFS_30.ipynb

## Objective

Evaluate sequential feature selection using different numbers of selected features for intrusion detection.
